# Project Description

Lokalin is a e-commerce showcasing traditional Indonesian products. This marketplace focuses on promoting local cultural goods including traditional foods (Makanan Khas Nusantara) and traditional crafts (Kerajinan Tradisional).

# How to Use
Prerequisites:
A modern web browser (Chrome, Firefox, Safari, Edge)
Basic understanding of web development if you plan to customize

Installation:
Clone or download the repository
git clone https://github.com/IkhsanMaulan4/Lokalin_SE.git
Or simply download the ZIP file and extract it to your preferred location.
